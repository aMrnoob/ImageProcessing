# 🧠 ImageProcessing App

Ứng dụng xử lý ảnh số sử dụng Streamlit, bao gồm các chức năng:
- 🎭 Nhận diện khuôn mặt
- 👁️ Nhận dạng đối tượng với YOLOv8
- 🍎 Nhận dạng trái cây
- 🔢 Nhận dạng chữ số viết tay (MNIST)
